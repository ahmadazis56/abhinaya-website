# Abhinaya Website

Complete Abhinaya Website with Next.js - Publisher, Creative, and Technology Solutions

## Features

- **Abhinaya Publisher** - Journal publishing and academic services
- **Abhinaya Creative** - Creative design and multimedia services  
- **Abhinaya Techno** - Technology solutions and software development
- **Admin Dashboard** - Complete content management system
- **Responsive Design** - Mobile-first approach with Tailwind CSS
- **Database Integration** - SQLite with Prisma ORM
- **Authentication** - Secure admin authentication system

## Technology Stack

- **Framework**: Next.js 16 with App Router
- **Styling**: Tailwind CSS v4
- **Database**: SQLite with Prisma ORM
- **Authentication**: NextAuth.js
- **Deployment**: Vercel (auto-deployment configured)

## Quick Start

1. Clone the repository
2. Install dependencies: `npm install`
3. Setup database: `npm run db:migrate`
4. Run development server: `npm run dev`
5. Open [http://localhost:3000](http://localhost:3000)

## Deployment

This project is configured for automatic deployment to Vercel. Any push to the `main` branch will automatically deploy to production.
